This thesis proposes a Personalized Language Learning (PLL) application that leverages \textbf{short-video} content (Reels) to enhance second language acquisition. The platform integrates authentic audiovisual materials, adaptive scaffolding, and cultural explanations to promote both linguistic competence and cultural understanding. Drawing on naturalistic and contextual learning principles, the system ensures repeated exposure to target vocabulary across meaningful contexts, guided by memory retention strategies such as the \textbf{Ebbinghaus forgetting curve}. Users can upload videos, which are automatically transcribed using an LLM-based API and curated for difficulty and relevance through a multi-stage \textbf{recommendation system} involving preprocessing, scoring, and re-ranking. The application is built with a React Native frontend, a Python-based backend, and a relational database, providing a cross-platform \textbf{mobile} experience. This work addresses gaps in current language learning platforms—including lack of authentic content, insufficient multimedia integration, and limited support for advanced learners—while combining educational value with engaging, gamified interactions inspired by hypercasual games. The project also builds upon prior coursework in entertainment software technologies and mobile application development, ensuring technical rigor and scalability.
